# TamagoishiiuWu
projet site internet

## Déploiement GitHub Pages
1. Allez dans **Settings → Pages**.
2. **Build and deployment** → *Deploy from a branch*.
3. **Branch** : `main` / **Folder** : `/root`.
4. Enregistrez. L'URL sera `https://<votre-pseudo>.github.io/tamagotchiEchomoriStyle`.

## Modifier le contenu
- Texte principal : `index.html`
- Couleurs/styles : `style.css`
- Logique des onglets : `script.js`
- Images : `assets/images/`
- Fichiers à télécharger : `assets/downloads/`
